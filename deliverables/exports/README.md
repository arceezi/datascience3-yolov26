# Export Outputs

Store generated export files in this folder.

This folder is reserved for converted output files such as:
- `article-draft.docx`
- `article-draft.pdf`
- `peer-review-statement.docx`
- `peer-review-statement.pdf`

Workflow reminder:
1. revise content in `manuscript/`
2. update the polished Markdown draft in `deliverables/`
3. export the Markdown draft to `.docx` or `.pdf`
4. save the generated files here
