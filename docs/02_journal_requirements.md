# 02 Journal Requirements

Style target:
- Scientific Reports

Formatting and writing constraints to keep visible during drafting:
- use standardized sections: Abstract, Introduction, Results, Discussion, Methods
- keep the abstract at 200 words maximum
- use Vancouver-style numbered references
- write in a technical, evidence-based tone
- explain not only what happened, but why the results likely behaved that way

Assignment-specific content requirements:
- include the comparative performance table for the three tested models
- include the confusion matrix and explain what it reveals
- include training dynamics such as loss or mAP curves
- discuss hyperparameter differences across the three setups
- describe the Google Colab and NVIDIA T4 environment
- describe dataset composition and augmentation
- mention the Ultralytics implementation steps
