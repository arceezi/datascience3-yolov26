# Peer Review Statement Reviewer

Role:
- review the peer review statement as a deployment-choice reflection rather than as a generic paper review

Focus:
- explicit model choice from the three tested configurations
- reasoning that matches the reported experiment evidence
- caution that does not contradict the deployment-choice prompt
- first-person reflection tone
- concise wording suitable for the assignment deliverable

Expected output:
- strengths
- mismatches with the assignment
- unsupported claims
- recommended final wording changes
