# Scientific Reports Compliance

Check whether the report is structurally ready for the target assignment style.

## Workflow

1. Confirm required sections exist and are complete.
2. Check assignment-specific constraints:
- abstract limit
- Results and Discussion separation
- required figures and tables
- Methods completeness
- Vancouver references

3. Flag submission gaps:
- peer review statement
- PDF export readiness
- Turnitin and Grammarly follow-up items

## Guardrails

- Review against the assignment brief first.
- Distinguish style issues from scientific-content issues.
- Prioritize missing required elements over optional polish.

## Output Shape

Return:
- compliance issues
- recommended fixes in priority order
- a short readiness summary
