# 00 Title And Metadata

## Working Title

Detection and Classification of Common Road Users in Philippine Traffic Using YOLOv26

## Title Revision Notes

### Brief Critique Of The Current Title

- `Detection and Classification` likely overstates the experiment because the study is centered on object detection, with class labels handled inside the detection task rather than as a separate classification pipeline.
- The title is understandable, but it is slightly long and can be made more journal-neutral.
- The current wording does not show that the paper is a comparative model-configuration study.
- The Philippine traffic context is useful and should remain because it helps define the application setting.

### Title Options

1. Detection of Common Road Users in Philippine Traffic Images Using YOLOv26
2. Object Detection of Common Road Users in Philippine Traffic Using YOLOv26
3. Comparative Evaluation of YOLOv26 Training Configurations for Detecting Common Road Users in Philippine Traffic
4. Detecting Common Road Users in Philippine Traffic Images: A Comparative YOLOv26 Study
5. YOLOv26-Based Detection of Common Road Users in Philippine Traffic Images

### Recommended Title

Detecting Common Road Users in Philippine Traffic Images: A Comparative YOLOv26 Study

### Recommendation Rationale

- removes the likely overstatement in `Classification`
- keeps the Philippine traffic context visible
- signals that the paper is a comparison study rather than a deployment paper
- reads more like a journal title while staying faithful to the current experiment

### Adoption Status

- current working title remains unchanged until the recommended title is explicitly accepted
